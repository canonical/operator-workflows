This charm provides an ingress integration, and can be integrated with any HTTP/HTTPS application. Below is a list of some of the known charms that currently integrate with it, but it may not be exhaustive:

* [content-cache-k8s](https://charmhub.io/content-cache-k8s)
* [discourse-k8s](https://charmhub.io/discourse-k8s)
* [gunicorn-k8s](https://charmhub.io/gunicorn-k8s)
* [hello-kubecon](https://charmhub.io/hello-kubecon)
* [indico](https://charmhub.io/indico)
* [mattermost-k8s](https://charmhub.io/mattermost-k8s)
* [wordpress-k8s](https://charmhub.io/wordpress-k8s)
